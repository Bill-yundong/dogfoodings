## 🔍 失败用例分析与修复建议

### A 类: 算法逻辑问题 (影响核心功能)

#### 1. 碳减排计算逻辑问题

**问题**: `carbonSaved` 计算结果为 0

**位置**: `src/utils/multiEnergyFlow.ts:172-173`

**当前逻辑**:
```typescript
const originalEmission = calculateCarbonEmission(station);
const adjustedEmission = calculateCarbonEmission(adjustedStation);
totalCarbonSaved += originalEmission - adjustedEmission;
```

**修复建议**:
当前优化可能导致碳排放上升，需要调整目标函数权重，增加碳排放惩罚因子：

```typescript
// 在目标函数中增加碳排放权重
return (
  0.25 * coolingDeviation +      // 降低制冷负荷偏差权重
  0.25 * heatingDeviation +      // 降低供热负荷偏差权重
  0.20 * electricityDeviation +  // 降低电力负荷偏差权重
  0.25 * emission +              // 增加碳排放权重 (原0.15)
  0.05 * (1 - efficiency)
);
```

---

#### 2. 约束条件满足性问题

**问题**: 部分优化结果 `constraintsSatisfied` 为 `false`

**位置**: `src/utils/multiEnergyFlow.ts:145-146`

**修复建议**:
在梯度下降迭代中增加约束边界检查，强制调整幅度在合理范围内：

```typescript
const newAdjustments = {
  cooling: Math.max(-constraints.maxCoolingAdjustment, 
           Math.min(constraints.maxCoolingAdjustment, 
           bestAdjustments.cooling - learningRate * coolingGrad)),
  heating: Math.max(-constraints.maxHeatingAdjustment, 
           Math.min(constraints.maxHeatingAdjustment, 
           bestAdjustments.heating - learningRate * heatingGrad)),
  electricity: Math.max(-constraints.maxElectricityAdjustment, 
               Math.min(constraints.maxElectricityAdjustment, 
               bestAdjustments.electricity - learningRate * electricityGrad)),
};
```

### B 类: 测试断言问题 (不影响功能, 仅测试修复)

#### 3. 文本匹配问题 - Emoji 分离

**问题**: 包含 emoji 的文本无法正确匹配，因为 emoji 在独立的 span 中

**影响测试用例**:
- 指挥中心主界面标题
- 碳排放监控标题
- 气象条件数据标题
- 多能互补态势标题
- 控制面板标题

**修复建议**: 使用 `findByText` 或正则表达式进行更灵活的匹配：

```typescript
// 修复前
expect(getByText('🏙️ 智慧城区能源指挥中心')).toBeInTheDocument();

// 修复后
expect(getByText(/智慧城区能源指挥中心/)).toBeInTheDocument();
```

---

#### 4. 重复文本匹配问题

**问题**: "正常"、"警告" 等状态文本在多个位置出现，导致 `getByText` 失败

**影响测试用例**:
- 不同状态能源站显示测试

**修复建议**: 使用 `getAllByText` 并检查数量：

```typescript
// 修复前
expect(getByText('正常')).toBeInTheDocument();
expect(getByText('警告')).toBeInTheDocument();

// 修复后
const normalTexts = getAllByText('正常');
expect(normalTexts.length).toBeGreaterThanOrEqual(3); // 3个站正常状态
const warningTexts = getAllByText('警告');
expect(warningTexts.length).toBe(1); // 1个站警告状态
```
