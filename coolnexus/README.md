# CoolNexus - 数据中心热管理平台

基于 Next.js 构建的数据中心散热场建模与热负荷分布实时同步系统。

## 核心功能

### 🔥 1. CFD 计算流体动力学引擎
- 异步 CFD 数值计算引擎
- 三维温场模拟与可视化
- 机柜气流短路风险检测
- 热空气回流分析

### 📊 2. 实时热负荷同步
- 运维平台与空调控制系统实时同步
- 机柜级热负荷分布监控
- 2秒级数据更新频率
- WebSocket 双向通信

### 🌡️ 3. 三维温场可视化
- Three.js 3D 点云温度场渲染
- 交互式视角控制（拖拽旋转、滚轮缩放）
- 机柜与空调位置实时映射
- 温度颜色渐变编码

### 💾 4. IndexedDB 历史数据存储
- 功耗快照持久化存储
- PUE 历史趋势分析
- 告警记录查询
- 支持 500+ 历史数据点

### ⚡ 5. PUE 能效管理
- 实时 PUE 指标计算
- 日/周/月平均统计
- 能效趋势分析（改进/恶化/稳定）
- 目标值对比与可视化

## 技术栈

- **框架**: Next.js 15 (App Router)
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **3D 渲染**: Three.js
- **本地存储**: IndexedDB (idb 库)
- **实时通信**: Socket.IO
- **数据可视化**: Recharts

## 核心模块说明

### 1. AsyncCFDEngine (CFD 计算引擎)
- 基于有限差分法的热传导计算
- 支持迭代过程进度回调
- 自动检测气流短路和热点风险
- 可配置网格分辨率和松弛因子

### 2. HeatLoadSyncManager (热负荷同步)
- 模拟实时数据同步
- 支持空调远程控制指令
- 事件驱动的数据更新
- 连接状态监控

### 3. IndexedDBStore (本地存储)
- 浏览器端持久化存储
- 自动清理旧数据（保留最近 500 条）
- 支持按时间范围查询
- 多对象存储（功耗、热负荷、告警）

### 4. TemperatureField3D (3D 可视化)
- 温度点云渲染
- 机柜和空调 3D 模型
- 鼠标交互控制
- 温度到颜色的实时映射

## 仪表盘功能

1. **PUE 指标卡片**
   - 当前 PUE 值
   - 趋势指示器
   - 日/周/月平均统计
   - 进度条可视化

2. **3D 温度场**
   - 实时温场可视化
   - CFD 计算触发按钮
   - 计算进度显示

3. **2D 热力图**
   - 机柜级温度分布
   - 鼠标悬停详情
   - 颜色图例说明

4. **风险告警**
   - 气流短路告警
   - 局部热点告警
   - 热空气回流告警
   - 严重程度分级显示

5. **精密空调状态**
   - 送风/回风温度
   - 制冷功率
   - 风扇转速
   - 运行状态
