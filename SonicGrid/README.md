# SonicGrid - 城市噪声污染空间分布量化系统

基于 React 的城市噪声污染可视化系统，通过声场物理建模实现声波在复杂建筑群中的反射、衍射衰减模拟。

## 核心功能

### 1. 声场物理建模
- **声波反射模拟**：基于几何声学原理，实现声波在建筑物表面的反射计算
- **衍射衰减模拟**：考虑不同频率声波在建筑边缘的衍射效应
- **空气吸收衰减**：基于温度、湿度、频率的空气吸收模型
- **几何扩散衰减**：实现球面波扩散衰减计算

### 2. 克里金插值热力图
- **变异函数模型**：支持高斯、球形、指数三种变异函数模型
- **空间插值**：基于采样点的克里金插值生成连续噪声分布
- **Canvas 实时渲染**：高性能热力图可视化

### 3. Web Audio 频谱分析 (Worker)
- **Web Worker 异步处理**：避免阻塞主线程
- **FFT 频谱分析**：实时音频频谱分析
- **RMS/峰值计算**：声级统计指标计算

### 4. IndexedDB 持久化存储
- **快照保存**：保存当前噪声分布快照
- **历史回放**：加载历史快照进行对比分析
- **时间范围查询**：按时间范围检索历史数据

## 技术栈

- **前端框架**: React 18 + TypeScript
- **构建工具**: Vite
- **存储方案**: IndexedDB (idb 库)
- **可视化**: Canvas API
- **并发处理**: Web Worker

## 使用说明

### 噪声源管理
- 支持四种噪声源类型：交通、工业、施工、社交
- 可调节声级 (40-120 dB)
- 可调节频率 (20-2000 Hz)
- 支持启用/禁用单个噪声源

### 建筑物管理
- 支持三种建筑材质：混凝土、玻璃、砖
- 不同材质具有不同的声反射系数
- 可添加/删除建筑物

### 快照功能
- 一键保存当前噪声分布状态
- 加载历史快照进行对比
- 删除不需要的快照记录

## 声场物理模型

### 反射系数
- 混凝土: 0.95
- 玻璃: 0.90
- 砖: 0.85

### 衰减计算
```
总衰减 = 几何扩散衰减 + 空气吸收衰减 + 反射衰减 + 衍射衰减
```

### 变异函数参数
- **块金值 (Nugget)**: 0.5
- **基台值 (Sill)**: 根据数据范围自适应
- **变程 (Range)**: 根据空间分布自适应
