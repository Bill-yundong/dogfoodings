# RelayLogic - 变电站继电保护装置监控系统

基于 Svelte 5 构建的变电站继电保护装置协同监控系统，实现了故障跳闸切片数据在设备运维与实时监控系统间的语义对齐，利用异步故障树演化模拟预测连锁故障风险，并配合 IndexedDB 存储万次故障录波的离线波形数据。

## 功能特性

### 1. 📊 实时监控面板
- 实时电压、电流、频率、温度监控
- 告警信息展示与确认
- 设备状态实时监测
- 系统日志记录
- 模拟故障注入功能

### 2. 🌳 故障树演化分析
- 标准故障树模型构建（基础事件、逻辑门、顶事件）
- 异步故障树演化模拟
- 连锁故障风险预测
- 节点状态可视化
- 风险等级评估与提示

### 3. 📈 故障录波查看器
- 基于 IndexedDB 的离线波形存储（支持万级数据量）
- 波形数据按设备、故障类型、时间范围查询
- 电压/电流波形可视化
- 波形数据导出功能
- 故障录波统计分析

### 4. 🔗 语义对齐模块
- 设备运维系统（中文）与实时监控系统（英文）数据对齐
- 统一语义标签映射
- 对齐置信度计算
- 可扩展的语义映射规则
- 对齐统计与历史记录

### 5. 🔧 设备运维管理
- 设备健康度评估与展示
- 设备参数与运行状态管理
- 预防性维护记录
- 维护历史跟踪
- 健康度趋势分析

## 技术架构

### 核心技术栈
- **前端框架**: Svelte 5
- **构建工具**: Vite
- **离线存储**: IndexedDB (idb)
- **数据可视化**: SVG 原生绘制

## 核心模块说明

### 数据模型 (FaultSlice)
- 统一的故障跳闸切片数据结构
- 支持设备信息、故障类型、严重程度、波形数据等多维度数据
- 提供便捷的序列化/反序列化方法

### 语义对齐服务 (SemanticAlignment)
- 双系统术语映射：中文 ↔ 统一语义 ↔ 英文
- 基于设备ID、故障类型、时间戳的多维对齐算法
- 可配置的对齐规则与阈值
- 置信度计算与评估

### 故障树模拟器 (FaultTreeSimulator)
- 异步演化模拟引擎
- 支持 AND/OR 逻辑门的故障传播计算
- 连锁故障风险预测算法
- 实时模拟状态可视化

### 波形存储服务 (WaveformStorage)
- IndexedDB 异步存储封装
- 支持按设备、时间、故障类型等多维度查询
- 自动存储配额管理（限制10000条）
- 数据导入/导出功能

## 使用说明

### 实时监控
1. 左侧导航点击"实时监控"
2. 查看系统运行参数与告警信息
3. 点击"模拟故障"按钮注入测试故障
4. 点击告警条目确认告警

### 故障树分析
1. 左侧导航点击"故障树分析"
2. 点击"开始模拟"按钮启动故障树演化
3. 查看节点状态变化与风险等级
4. 点击"风险预测"查看连锁故障影响范围

### 波形查看
1. 左侧导航点击"故障录波"
2. 点击"生成示例数据"创建测试波形数据
3. 左侧列表选择查看波形详情
4. 查看电压/电流波形可视化

### 语义对齐
1. 左侧导航点击"语义对齐"
2. 编辑左右两侧的 JSON 数据
3. 点击"执行语义对齐"查看对齐结果
4. 查看统一语义标签与置信度

### 设备运维
1. 左侧导航点击"设备运维"
2. 点击设备卡片查看详情
3. 点击"执行预防性维护"更新设备健康度
4. 查看维护历史记录

## 特色功能

1. **跨系统语义对齐**：解决异构系统间数据标准不统一问题
2. **异步故障演化模拟**：基于故障树的连锁故障风险预测
3. **大规模离线存储**：基于 IndexedDB 的万级波形数据存储
4. **低侵入式集成**：可与现有电力监控系统无缝对接
5. **响应式界面**：支持多种屏幕尺寸与设备
