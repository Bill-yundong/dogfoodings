## 1. 产品概述

精密磨削表面粗糙度预测与生产数字化复刻系统，基于 Next.js 构建，实现加工功率谱数据在生产执行与质量反馈系统间的实时对齐。通过异步分形几何分析算法提取表面形貌特征，优化加工进给参数，并利用 IndexedDB 缓存关键零件的加工指纹快照，实现精密零部件生产的全流程数字化复刻。

- **核心目标**：通过 AI 预测表面粗糙度，优化加工参数，实现精密磨削过程的数字化闭环控制
- **目标用户**：精密制造工程师、质量检测人员、生产管理人员
- **市场价值**：提升精密零部件加工良率、降低试错成本、实现生产过程可追溯

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 工艺工程师 | 系统账号 | 配置加工参数、查看预测结果、优化进给参数 |
| 质量检测员 | 系统账号 | 录入实测数据、查看质量报告、对比预测精度 |

2. **粗糙度预测中心**：功率谱数据分析、分形几何特征提取、粗糙度 AI 预测
3. **参数优化引擎**：进给参数优化建议、加工参数对比、参数历史回溯
4. **加工指纹库**：IndexedDB 缓存管理、零件快照查询、数字化复刻对比
5. **系统设置**：数据源配置、阈值参数调整、数据导出管理

## 3. 核心流程

```mermaid
flowchart LR
    A["生产系统<br/>功率谱数据"] --> B["实时数据对齐<br/>时间戳同步"]
    B --> C["异步分形分析<br/>特征提取"]
    C --> D["AI模型预测<br/>表面粗糙度"]
    D --> E["参数优化引擎<br/>进给参数建议"]
    E --> F["加工执行<br/>参数调整"]
    F --> G["质量反馈系统<br/>实测数据录入"]
    G --> H["IndexedDB<br/>加工指纹快照"]
    H --> I["数字化复刻<br/>全流程追溯"]
    G --> C
```

**核心流程描述**：
1. 生产系统实时采集磨削功率谱数据，与质量反馈系统进行时间戳对齐
2. 异步分形几何分析算法处理功率谱数据，提取表面形貌特征
3. AI 模型基于特征参数预测表面粗糙度并给出置信区间
4. 参数优化引擎根据预测结果生成进给参数优化建议
5. 加工执行系统应用优化参数，实测数据录入质量反馈系统
6. 关键零件加工指纹快照存入 IndexedDB，支持全流程数字化复刻与追溯

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：工业蓝 `#0F52BA`，代表精密与专业
- **辅助色**：科技青 `#00D4AA` 用于数据可视化高亮，警示橙 `#FF6B35` 用于异常提醒
- **背景色**：深空灰 `#0D1117` 搭配渐变网格纹理，营造工业科技感
- **按钮风格**：微立体圆角按钮，悬浮时发光效果
- **字体**：标题使用 `Space Grotesk` 彰显科技感，正文使用 `Inter` 保证可读性
- **布局风格**：深色仪表盘布局，卡片式模块化设计，数据可视化优先

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 实时监控面板 | 功率谱波形图 | Canvas 实时波形渲染、多频段频谱分析、时间轴控制 |
| 实时监控面板 | 状态概览卡片 | 玻璃拟态卡片、渐变进度条、实时数值动画 |
| 粗糙度预测 | 特征提取面板 | 分形维数可视化、特征雷达图、异步加载进度指示 |
| 粗糙度预测 | 预测结果展示 | 预测区间柱状图、实测对比散点图、置信度指示环 |
| 参数优化 | 优化建议引擎 | 参数滑块对比、热力图展示参数敏感度、应用确认动效 |
| 加工指纹库 | 零件快照列表 | 时间线布局、卡片悬浮放大、搜索过滤动效 |
| 加工指纹库 | 数字化复刻视图 | 双栏对比布局、差异高亮标注、参数映射连线 |

### 4.3 响应式设计
- **桌面端优先**：支持 1920×1080 及以上分辨率，充分利用宽屏展示多维度数据
- **平板适配**：1024px 断点，侧边栏收起为图标菜单，图表自适应缩放
- **移动端**：768px 断点，卡片单列布局，核心指标优先展示，图表支持触摸交互

### 4.4 数据可视化指引
- **功率谱图**：使用 WebGL 加速渲染，支持百万级数据点实时更新
- **分形分析图**：3D 表面形貌可视化，支持旋转缩放查看细节
- **预测结果**：误差带填充的折线图，直观展示预测置信区间
- **参数热力图**：颜色编码展示不同参数组合的预测效果
- **动画效果**：数据更新时采用平滑过渡动画，避免视觉跳变
