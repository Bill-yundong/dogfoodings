## 1. 产品概述

知识链路（KnowledgeLink）——面向深度阅读者的量化知识体系成长路径引擎。基于知识图谱演化数据实现阅读库、笔记系统与复习引擎间的增量对齐，利用异步认知负荷预测模型优化间隔重复计划，配合 IndexedDB 实现万卷藏书笔记的离线增量索引，支撑大规模知识库的跨场景无缝流转。

- **核心问题**：知识碎片化、阅读-笔记-复习割裂、遗忘曲线不可控、大规模知识库检索延迟高
- **目标用户**：深度阅读者、学术研究者、终身学习者
- **产品价值**：将阅读行为转化为可量化的知识资产，通过认知负荷感知的间隔重复算法实现长期记忆留存率 ≥ 85%

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 学习者 | 本地初始化（无服务端） | 全部功能：阅读、笔记、复习、图谱 |

### 2.2 功能模块

1. **阅读库**：书籍/文章管理、阅读进度追踪、知识摄入量化
2. **笔记系统**：结构化笔记、知识关联标注、双向链接
3. **复习引擎**：间隔重复调度、认知负荷预测、记忆曲线可视化
4. **知识图谱**：图谱演化可视化、跨场景增量对齐、知识成长路径
5. **成长仪表板**：量化指标面板、知识覆盖率热力图、学习节奏分析

## 3. 核心流程

### 3.1 知识摄入→内化→强化闭环

用户在阅读库中摄入知识 → 标注生成笔记 → 笔记自动抽取为复习卡片 → 复习引擎根据认知负荷调度复习 → 复习结果反馈至知识图谱更新节点强度 → 图谱演化驱动增量对齐同步至阅读库与笔记系统。

```mermaid
flowchart TD
    "阅读库" -->|"标注+笔记"| "笔记系统"
    "笔记系统" -->|"抽取卡片"| "复习引擎"
    "复习引擎" -->|"负荷预测+调度"| "间隔重复计划"
    "间隔重复计划" -->|"复习反馈"| "知识图谱"
    "知识图谱" -->|"增量对齐"| "阅读库"
    "知识图谱" -->|"增量对齐"| "笔记系统"
    "知识图谱" -->|"演化数据"| "成长仪表板"
```

### 3.2 离线增量索引流程

```mermaid
flowchart TD
    "用户操作" -->|"写入事件"| "事件队列"
    "事件队列" -->|"异步批处理"| "增量索引器"
    "增量索引器" -->|"写入"| "IndexedDB"
    "IndexedDB" -->|"全量/增量查询"| "搜索服务"
    "搜索服务" -->|"结果"| "各页面"
```

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深靛蓝 `#1a1a2e` 为底，琥珀金 `#e2b714` 为强调，营造"智慧殿堂"氛围
- **辅助色**：雾蓝 `#16213e`、暗紫 `#0f3460`、薄荷绿 `#53d769`（状态指示）
- **按钮风格**：圆角 8px，微妙渐变，hover 时发光效果
- **字体**：标题用 `Playfair Display`（学术质感），正文用 `Source Sans 3`（清晰可读），代码/数据用 `JetBrains Mono`
- **布局风格**：左侧导航栏 + 主内容区，卡片式模块布局
- **图标**：Lucide 图标库，线描风格
- **动效**：页面切换淡入，卡片悬浮微动，图谱节点脉冲动画

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|

### 4.3 响应式设计

- 桌面优先设计（1440px 基准）
- 平板适配（768px）：侧边栏折叠为抽屉，图谱缩放交互
- 移动端适配（375px）：底部标签导航，卡片单列，复习全屏模式

### 4.4 动效设计

- 页面加载：交错渐入（stagger fade-in），延迟 80ms 递增
- 图谱节点：呼吸脉冲动画，强度映射节点权重
- 复习翻转：3D 翻转动画（rotateY 180deg），时长 400ms
- 进度条：流光渐变动画
- 热力图：色块渐变过渡
