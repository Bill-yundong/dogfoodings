## 1. 产品概述
基于 React 的 WMS 自动化货位分配与入库系统，实现大型配送中心的智能仓储管理。通过优化算法提升货位利用率，实现 WMS 系统与堆垛机控制模块的实时联动，支撑万级 SKU 的高频协同作业。

- **核心价值**：提升仓储作业效率 30%+，货位利用率提升至 85% 以上
- **目标用户**：仓储管理员、物流调度员、系统运维人员
- **应用场景**：电商配送中心、制造业原料仓、第三方物流仓储

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 系统管理员 | 系统预置 | 全功能访问、参数配置、用户管理 |
| 仓储管理员 | 管理员创建 | 货位监控、入库调度、报表查看 |
| 运维工程师 | 管理员创建 | 算法参数调整、碎片整理操作、日志查看 |

### 2.2 功能模块
1. **智能仪表盘**：实时监控仓储状态、作业效率、货位利用率
2. **货位分配中心**：自动化货位分配、SKU 关联分析、入库调度
3. **堆垛机控制**：设备状态监控、任务队列、实时联动反馈
4. **空间碎片管理**：碎片检测、异步整理、整理效果评估
5. **SKU 流动性分析**：流动性快照、冷热 SKU 识别、趋势预测

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：深空蓝 (#0F172A) 作为背景，科技蓝 (#3B82F6) 作为主色，翡翠绿 (#10B981) 表示正常，琥珀橙 (#F59E0B) 表示警告，玫瑰红 (#F43F5E) 表示告警
- **字体**：展示字体使用 Orbitron 增强科技感，正文字体使用 JetBrains Mono 保证数据可读性
- **布局**：深色工业风仪表盘设计，网格化布局，卡片式模块，突出数据可视化
- **动效**：数据加载骨架屏、数字滚动动画、状态切换过渡、告警脉冲效果

### 4.2 页面设计概述
| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|----------|
| 综合仪表盘 | 实时概览 | 大号 KPI 卡片、趋势折线图、环形进度图、告警列表 |
| 综合仪表盘 | 货位热力图 | 三维货架可视化、颜色编码热力、交互式缩放、货位详情弹窗 |
| 货位分配 | 入库作业 | 表单式入库单、策略选择器、分配结果预览、执行进度条 |
| 货位分配 | 关联分析 | 关联矩阵热力图、SKU 关联网络图、推荐列表 |
| 堆垛机监控 | 设备状态 | 设备卡片列表、运行动画、状态指示灯、任务进度 |
| 空间管理 | 碎片检测 | 碎片化程度仪表盘、碎片位置标记、优先级排序 |
| 空间管理 | 碎片整理 | 整理任务列表、进度追踪、前后对比图 |
| SKU 分析 | 流动性快照 | 数据表格、搜索过滤、分类标签、分页器 |

### 4.3 响应式
- 桌面端优先设计，支持 1920×1080 及以上分辨率
- 平板端自适应布局，侧边栏可折叠
- 移动端保留核心监控功能，简化操作界面

### 4.4 可视化设计
- 货位热力图使用 Canvas 渲染万级货位
- 实时数据流使用 WebSocket 模拟推送
- 图表使用 Recharts 实现交互式可视化
