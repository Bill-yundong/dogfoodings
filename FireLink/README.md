# 🔥 FireLink - 智能火灾逃生指引系统

基于 SolidJS 的大型商场火灾逃生指引系统，实现实时烟雾浓度场数据在监控中心与智能导引终端间的协同映射。

## ✨ 功能特性

### 核心功能
- **实时烟雾浓度场模拟** - 基于扩散模型的动态烟雾浓度场可视化
- **异步启发式路径优化** - A* 算法变体，动态避开高风险区域
- **IndexedDB 离线缓存** - 楼宇各层静态路网的离线切片存储
- **双模式界面** - 监控中心大屏 + 智能导引终端
- **断电模式支持** - 极端断电环境下的本地化疏散引导

### 技术亮点
- 🚀 **SolidJS** - 高性能响应式前端框架
- 🗄️ **IndexedDB** - 离线数据缓存与同步队列
- 🧭 **A* 路径规划** - 带风险权重的启发式搜索
- 🌊 **烟雾扩散模型** - 实时浓度场模拟与可视化
- 🎨 **Tailwind CSS** - 现代化 UI 样式
- 📱 **响应式设计** - 适配多终端显示

## 🎮 使用说明

### 监控中心模式
1. 点击 **「触发紧急状态」** 启动烟雾模拟
2. 在地图上点击任意节点计算疏散路径
3. 左侧面板可切换楼层、控制显示选项
4. 右侧面板查看实时数据和统计信息
5. 可切换电源状态和离线模式进行测试

### 智能终端模式
1. 选择当前所在楼层
2. 在地图上点击您的位置
3. 点击 **「计算逃生路线」** 获取最优路径
4. 按照右侧指引逐步撤离
5. 可展开详细的步行方向指引

## 🧠 核心算法

### 路径优化算法
- **启发函数**: 欧几里得距离 + 风险权重
- **代价函数**: 距离 × 0.4 + 风险 × 0.6
- **多路径支持**: 生成 3 条备选路径
- **动态更新**: 烟雾浓度变化时自动重算

### 烟雾扩散模型
- **网格大小**: 10×10 单位
- **扩散速率**: 0.15/秒
- **衰减速率**: 0.01/秒
- **浓度范围**: 0-100%

## 📊 数据结构

### 楼层地图
- 3 层楼的商场布局
- 每层包含节点、边、房间、墙体
- 支持跨楼层路径规划（楼梯连接）

### 节点类型
- 🚪 **exit** - 安全出口
- 🪜 **stair** - 楼梯间
- 🛗 **elevator** - 电梯（火灾禁用）
- 🏠 **room** - 房间/商铺
- ➡️ **corridor** - 走廊节点
- 🔀 **junction** - 交叉路口

## 🔧 配置说明

### 路径规划配置
```typescript
{
  walkingSpeed: 1.4,           // 步行速度 (m/s)
  riskWeight: 0.6,             // 风险权重
  distanceWeight: 0.4,         // 距离权重
  maxSearchIterations: 10000   // 最大搜索迭代
}
```

### 烟雾模拟配置
```typescript
{
  gridSize: 10,                // 网格大小
  diffusionRate: 0.15,         // 扩散速率
  maxConcentration: 100,       // 最大浓度
  decayRate: 0.01              // 衰减速率
}
```

## 🛡️ 安全特性

1. **离线模式** - IndexedDB 缓存地图数据，断网时仍可工作
2. **断电模式** - 低功耗运行，保留核心导引功能
3. **风险分级** - 5 级风险评估，自动避开危险区域
4. **多路径冗余** - 提供多条逃生路线备选
