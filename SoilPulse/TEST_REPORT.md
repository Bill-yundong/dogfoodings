## 🔧 测试环境与配置

### 技术栈

```
Jest 29.x          → 测试运行器

fake-indexeddb 5.x → IndexedDB 模拟
TypeScript 5.x     → 类型系统
Node.js 20.x       → 运行环境
```

### Jest 配置要点

1. **测试环境**: `jsdom` - 模拟浏览器环境
2. **模块映射**: 支持 `@/` 路径别名
3. **覆盖率收集**: 自动收集核心库文件覆盖率
4. **超时设置**: 10秒（适应性能测试）

### 测试数据特点

1. **样本多样性**: pH范围4.0-9.0，覆盖正常和极端土壤
2. **地理分布**: 模拟不同经纬度的田块
3. **养分梯度**: 养分含量从极低到较高的完整梯度
4. **规模验证**: 从单田块到批量田块的可扩展性
