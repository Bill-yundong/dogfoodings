## 五、数据流转完整链路测试

### 5.1 排放数据全链路

```
用户输入排放数据
    ↓
[Emissions.vue] 表单提交
    ↓
[carbon.ts] addEmissionRecord()
    ├─ 计算排放量 (quantity * factor)
    ├─ 构建 CarbonRecord 对象
    ├─ records.value.unshift() → 响应式更新列表
    └─ loadAggregatedData() → 更新统计数据
        ↓
[Dashboard] 实时更新所有图表
    ├─ 总排放量数字
    ├─ Scope 1/2/3 占比
    ├─ 趋势图表
    └─ 部门分布图表
```

**测试结果**: ✅ 完整链路正常，数据从输入到展示无断点

### 5.2 LCA 分析全链路

```
用户配置LCA参数
    ↓
[LCA.vue] submitLCA() 调用
    ↓
[carbon.ts] startLCACalculation()
    ├─ 调用 lcaEngine.addJob() 生成 jobId
    ├─ 添加监听器监听进度
    └─ 等待 complete 事件
        ↓
[lcaEngine.ts] 异步计算流程
    ├─ 任务队列排队
    ├─ 5 阶段依次计算
    ├─ 每阶段 200ms 进度更新
    └─ 完成时触发 notifyUpdate('complete')
        ↓
[Store] 接收结果
    ├─ 推送至 lcaCalculations 数组
    └─ currentLcaJob 置空
        ↓
[LCA.vue] 列表自动更新
    └─ 点击查看详情 → 对话框展示完整数据
```

**测试结果**: ✅ 完整异步链路正常，进度实时反馈，结果持久化可查看
