# OrbitLink 空间碎片交会碰撞预警系统 - 产品需求文档（PRD）

## 1. 产品概述
OrbitLink 是一个面向科研分析与航天器控制模块的空间近距离会合动力学可视化与语义对齐平台。通过在浏览器中构建高保真的轨道力学仿真环境，实现万级编目碎片的实时交会预警、碰撞概率评估以及轨道档案检索。
- 主要目标用户：航天任务规划师、空间态势感知（SSA）研究员、航天器 GNC 工程师。
- 核心价值：在前端完成异步多线程的星历增量推演与碰撞概率预测，构建不依赖后端的高并发空间安全分析中枢。

## 2. 核心功能

### 2.1 功能模块
1. **轨道推演中枢（Propagation Lab）**：以 6 轨道根数驱动的二体问题解析模型 + J2/J3 摄动修正，支持时间步长可调的可视化推演。
2. **近距离会合扫描（Conjunction Scan）**：对目标航天器与编目碎片进行 TCA（最近时刻）搜索，输出距离阈值内的事件清单。
3. **碰撞概率评估（Pc Assessment）**：基于位置协方差的 3D 高斯积分近似（Foster 方法），输出 Pc 与 Mahalanobis 距离。
4. **碎片档案中心（Catalog Archive）**：IndexedDB 中持久化万级碎片的 NORAD ID、TLE、轨道周期、RCS 等档案，并提供检索与过滤。
5. **3D 态势显示（SpaceView）**：基于 @react-three/fiber 的地心惯性系视图，支持多轨道同时渲染、时间轴播放、碰撞事件闪烁。

### 2.2 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 态势总览 | 顶部指标条 | 实时显示活动轨道数、近期会合事件数、Pc 超阈值数、平均计算延迟 |
| 态势总览 | 3D 视图 | 展示地球 + 选中目标轨道 + 所有候选碎片轨道，可暂停/播放/拖动时间滑块 |
| 态势总览 | 会合事件列表 | 按 TCA 升序展示事件，支持按 Pc/距离/相对速度排序与过滤 |
| 态势总览 | 目标详情面板 | 显示目标航天器 6 根数、位置速度、协方差矩阵、TLE |
| 碎片档案 | 检索面板 | 支持按 NORAD ID、轨道类型（LEO/MEO/GEO）、RCS 大小过滤 |
| 碎片档案 | 碎片详情 | 展示单条碎片的 TLE、轨道周期、近地点/远地点、倾角、历史 TCA |
| 仿真实验室 | 参数配置 | 可调积分步长、摄动阶数、碰撞半径、Pc 阈值 |
| 仿真实验室 | 计算监控 | 显示 Worker 池状态、单批次耗时、吞吐量（ephemeris/s） |

## 3. 核心流程
```mermaid
flowchart LR
  A["用户选择目标航天器"] --> B["从 IndexedDB 加载碎片档案"]
  B --> C["分发到 Web Worker 池"]
  C --> D["异步计算每颗碎片的星历增量"]
  D --> E["执行近距离会合扫描，记录 TCA 与最小距离"]
  E --> F["计算碰撞概率 Pc"]
  F --> G["3D 视图 + 事件列表刷新"]
  G --> H{"Pc 超阈值?"}
  H -- "是" --> I["发出红色告警并持久化到历史记录"]
  H -- "否" --> J["更新统计指标"]
```

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：深空黑 `#05070d`，次色：轨道青 `#38bdf8`、告警红 `#ef4444`、数据金 `#facc15`。
- 按钮风格：扁平 + 下边框高亮（`border-b border-cyan-400/40`），hover 时发光。
- 字体：标题使用 `Space Grotesk` / 系统 monospace 组合，正文使用 `Inter`。
- 布局：左侧导航 + 主画布 + 右侧数据面板的三栏指挥中心布局。
- 图标风格：Lucide 线性图标，配合单色填充。

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI 元素 |
|---------|---------|---------|
| 态势总览 | 顶部指标条 | 4 个数字卡片，背景 `#0b1220`，带顶部渐变高光 |
| 态势总览 | 3D 视图 | 全屏画布 + 右下时间滑块 + 左上轨道图例 |
| 态势总览 | 会合事件列表 | 暗色表格，悬停行背景 `#1e293b`，Pc 超阈值行红色描边 |
| 碎片档案 | 检索面板 | 左侧筛选芯片 + 右侧分页表格 |
| 仿真实验室 | 计算监控 | 进度条 + Worker 线程格网 + 吞吐量曲线 |

### 4.3 响应式
桌面端优先（≥1280px），1024~1280 两栏堆叠，<1024 单栏纵向布局。
