## 1. 产品概述

浅层地源热泵热平衡稳定性分析平台 - 面向长寿命低碳采暖运维的跨系统运营支撑工具。
- 主要目的：分析地源热泵系统的热平衡稳定性，预测土壤热透支风险，实现换热效率数据在运维与建筑节能系统间的语义同步
- 目标用户：暖通工程师、能源管理专家、建筑运维人员
- 核心价值：为低碳采暖系统提供科学化、数据化的运营决策支持

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 系统管理员 | 后台创建 | 全系统管理、用户管理、数据配置 |
| 暖通工程师 | 邀请注册 | 热平衡分析、热漂移预测、数据查询 |
| 能源管理专家 | 邀请注册 | 换热效率分析、跨系统数据同步、报告生成 |

### 2.2 功能模块

1. **系统首页**：实时监控面板、系统健康状态、快速导航
2. **热平衡分析页**：土壤热平衡计算、换热效率分析、稳定性评估
3. **热漂移预测页**：异步热漂移演化模型、土壤热透支预测、趋势分析
4. **换热孔管理页**：万级换热孔数据管理、历史地温快照查询、IndexedDB 离线存储
5. **数据同步页**：系统运维与建筑节能系统间的语义同步、数据映射配置
6. **系统设置页**：用户管理、参数配置、API 密钥管理

## 3. 核心流程

用户登录系统后，首先进入系统首页查看整体运行状态。暖通工程师可进入热平衡分析页进行土壤热平衡计算和稳定性评估。能源管理专家可使用热漂移预测页进行土壤热透支预测分析。换热孔管理页提供万级数据的高效查询和历史地温快照回放，数据通过 IndexedDB 实现本地缓存。数据同步页实现换热效率数据在系统运维与建筑节能系统间的语义同步。

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深青色 (#0D9488) 代表地球与能源
- 辅助色：橙色 (#F97316) 代表热量与预警
- 中性色：深灰 (#1F2937) 背景，浅灰 (#F3F4F6) 内容区
- 按钮样式：圆角设计，渐变背景，微动效
- 字体：系统默认字体栈，标题加粗
- 布局风格：卡片式布局，侧边导航，顶部状态栏
- 图标风格：线性图标，lucide-react

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|----------|
| 系统首页 | 实时监控面板 | 四列指标卡片，渐变背景，数据动画 |
| 系统首页 | 系统健康状态 | 状态指示点，脉冲动画，颜色编码 |
| 热平衡分析页 | 土壤热平衡计算 | 表单输入区，计算结果展示，图表可视化 |
| 热漂移预测页 | 热漂移演化模型 | 3D 或 2D 热力图，参数滑块，预测曲线 |
| 换热孔管理页 | 换热孔数据管理 | 数据表格，分页，搜索筛选，虚拟滚动 |
| 换热孔管理页 | 历史地温快照 | 时间轴控制器，温度曲线，IndexedDB 状态指示 |

### 4.3 响应式设计

- 桌面优先设计，最小支持 1280px 宽度
- 平板端自适应布局，卡片堆叠显示
- 移动端基础支持，核心功能可用
