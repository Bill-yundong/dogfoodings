## 1. 产品概述

AirCargoNexus 是一个基于 Svelte 5 的航空货运装载平衡（Weight & Balance）前端工程，用于在货站配载系统与机组配平终端之间实现业务对齐。通过异步分支定界启发式算法在前端极速解算最佳三维装箱装载矩阵，以降低飞行油耗；并利用 IndexedDB 实现万级货物时空配载快照的持久化存储。

- 目标用户：货站配载员、机组飞行员、运行控制中心（OCC）
- 解决问题：货运航班装载空间利用率低、重心偏移导致油耗上升、配载与机组信息不同步
- 核心价值：将装载计划解算时间压缩到毫秒级，提供可回放的时空快照，实现地-空配载业务闭环

## 2. 核心功能

### 2.1 用户角色

| 角色 | 登录方式 | 核心权限 |
|------|----------|----------|
| 货站配载员 | 工号 / 离线模式 | 创建航班、录入货物、触发解算、导出配载单 |
| 机组飞行员 | 工号 / 离线模式 | 查看航班配平、微调、锁定装载方案 |
| OCC 调度 | 工号 | 多航班总览、偏差告警、快照回放 |

### 2.2 功能模块

1. **航班装载工作台**：航班选择、ULD（集装器）舱位布局、货物清单、重心包线叠加显示
2. **三维装载可视化**：等距/透视视图展示舱内 ULD 与散货位置，实时显示重心矢量
3. **解算控制台**：算法参数配置、分支定界进度、候选方案对比、最优装载矩阵输出
4. **配载快照中心**：IndexedDB 快照列表、时间轴回放、差异对比、导入/导出 JSON
5. **机组配平终端**：机组视角简洁版、配平微调、燃油影响估算、方案锁定同步

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|

## 3. 核心流程

1. 配载员选择航班，录入或导入货物清单（ULDs + 散货）
2. 配置解算参数（时间预算、重心容差、油耗权重）
3. 点击"开始解算"→ 前端 Web Worker 执行异步分支定界启发式算法
4. 实时展示进度与候选方案，达到时间上限或最优解后输出装载矩阵
5. 配载员在 3D 视窗中查看装载方案与重心矢量
6. 方案被锁定后，生成配载快照并存入 IndexedDB
7. 机组终端拉取同一航班的最新快照，进行配平确认或微调
8. OCC 可在任意时刻回放历史快照，进行偏差分析


## 4. 用户界面设计

### 4.1 设计风格

- 主色：深航空蓝 `#0B2545`，辅色：琥珀金 `#D4A017`、青绿 `#2EC4B6`
- 语义色：正常绿、警告黄、告警红
- 按钮：扁平矩形、圆角 4px、悬停上浮阴影
- 字体：显示字 IBM Plex Sans SemiBold、正文字 IBM Plex Sans、等宽字 JetBrains Mono（用于参数与数据）
- 布局：顶部导航 + 左侧货物列表面板 + 中央 3D 视窗 + 右侧参数/重心面板
- 图标：线性图标，与航空货运语义一致（ULD、翼型、重心、燃油罐）

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 航班装载工作台 | 顶栏 | 品牌 Logo、航班号胶囊、状态标签、用户头像 |
| 航班装载工作台 | 左侧货物清单 | 卡片列表、重量/体积徽章、危险品图标、拖拽排序 |
| 航班装载工作台 | 中央 3D 视窗 | 货舱骨架、ULD 立方体、重心箭头、剖切按钮 |
| 航班装载工作台 | 右侧重心面板 | 包线图、MAC% 刻度、配平建议列表 |
| 解算控制台 | 参数面板 | 滑块（时间、权重）、下拉（策略）、数值输入 |
| 解算控制台 | 进度面板 | 折线图（上界/下界）、数值徽章、剪枝率 |
| 配载快照中心 | 时间轴 | 可拖动轨道、快照节点、差异高亮 |
| 机组配平终端 | 配平面板 | 大号重心指示器、配平滑块、节省燃油数字 |

### 4.3 响应式

- 桌面优先（≥1280px）：四栏布局
- 平板（768~1279px）：货物面板与重心面板折叠为抽屉
- 移动端（<768px）：单列，3D 视窗为主，其他面板通过底部 Tab 切换
- 机组终端另提供大字号、高对比度模式

### 4.4 3D 场景指引

- 环境：中性灰渐变背景 + 微弱网格地面，突出货舱
- 光照：主光 Key Light 右上方、补光 Fill Light 左下方，边缘光 Rim Light 勾勒 ULD
- 相机：透视相机，默认等距视角，可通过鼠标拖拽旋转、滚轮缩放
- 构图：货舱骨架居中，重心箭头叠加于几何中心并随解算结果更新
- 交互：ULD 可悬停高亮，点击查看详细属性；剖切平面拖动查看层次
- 后处理：Bloom 仅作用于重心箭头与告警标记，保持科技感但不喧宾夺主
- 资产：程序化生成 ULD 立方体与货舱骨架，避免外部依赖，性能预算 ≤ 50k 顶点
